title: Welcome to Dria
---

Dria builds infrastructure for developers who need reliable inference at scale. This site covers two capabilities that share the same auth stack and operational tooling:

- **Batch Inference** &mdash; high-throughput, low-cost `.jsonl` job execution
- **GEPA** &mdash; automated prompt evolution using the Genetic-Pareto method

Use this documentation as a workflow guide:

1. Start with the project overview to understand concepts and service limits.
2. Walk through the task-specific guides (preparing input files, monitoring jobs, etc.).
3. Jump to the autogenerated **API Reference** in each section whenever you need exact request and response schemas.

Both services use the `https://mainnet.dkn.dria.co/api/v0` base URL and the same `x-api-key`. Rotate keys regularly and scope keys to automation where possible.
