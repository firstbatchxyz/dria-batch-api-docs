## Supported Models

GEPA supports the following models:

- **gpt-5-mini** - Latest GPT-5 Mini (fast and cost-effective)
- **gpt-5** - Latest GPT-5 (highest quality)
- **gpt-4.1** - GPT-4.1
- **gpt-4o** - GPT-4o
- **gpt-4o-mini** - GPT-4o Mini (balanced performance and cost)
