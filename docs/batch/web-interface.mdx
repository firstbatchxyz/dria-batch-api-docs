title: Web Interface
---

Prefer buttons over scripts? The [Batch web interface](https://dria.co/batch-inference) mirrors every API workflow while handling auth, uploads, and monitoring for you.

## What You Can Do

- Upload `.jsonl` files from your desktop (the UI still uses the pre-signed URL flow behind the scenes).
- Track job progress with live status bars and automatic refresh.
- Download output files directly when a job finishes.
- Generate or rotate API keys without leaving the console.

## Suggested Workflow

1. Draft and validate your `.jsonl` locally.
2. Log in with the same account used for API keys.
3. Drag-and-drop the file into the uploader and optionally add metadata tags.
4. Watch the status card. You can close the page; the job continues server-side.
5. Download the results once the UI shows `Completed` or copy the job ID to use with the API later.

Behind the scenes the UI uses the same endpoints documented in the autogenerated **Batch API Reference**, so you can always switch from UI to code without surprises.
