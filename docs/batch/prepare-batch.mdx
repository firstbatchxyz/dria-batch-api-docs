## Input Schema

Create a `.jsonl` file where **each line is an independent inference request**. Every object must contain:

| Field | Description |
| --- | --- |
| `custom_id` | Unique identifier that lets you join outputs back to inputs. We recommend UUID v4. |
| `type` | Task type. Today `completions` is supported for chat-style payloads. |
| `version` | API version. Use `v1` unless otherwise announced. |
| `body` | Standard OpenAI-style request body: `model`, `messages`, `max_tokens`, `temperature`, etc. |

Example:

```jsonl
{"custom_id": "7278c795-19fb-48cc-b9bd-7961904d1a8c", "type": "completions", "version": "v1", "body": {"model": "gemma3:12b", "max_tokens": 1024, "messages": [{"role": "user", "content": "Problem description..."}]}}
{"custom_id": "fec80e3d-a1d4-445c-860e-87aa1fab4281", "type": "completions", "version": "v1", "body": {"model": "gemma3:12b", "max_tokens": 1024, "messages": [{"role": "user", "content": "Another prompt..."}]}}
```

## Best Practices

- **Keep lines self-contained.** Avoid referencing shared state across lines; jobs can be sharded to different workers.
- **Ship metadata in `body.metadata`.** This shows up in job results and makes debugging easier.
- **Chunk large imports.** For millions of tasks, create multiple files and launch them sequentially to avoid upload retries.
- **Stay within limits.** See [Limits & Models](docs/batch/limits) for the latest file size and per-plan constraints.
- **Encrypt before upload** if your data is sensitive - pre-signed URLs go straight to S3, so client-side encryption is supported.

## Validate Before Upload

Use a quick script to catch malformed lines locally. The snippet below asserts schema requirements, checks line length, and ensures unique IDs.

```python
import json, uuid, pathlib

file_path = pathlib.Path("batch.jsonl")
seen = set()
for i, line in enumerate(file_path.read_text().splitlines(), start=1):
    obj = json.loads(line)
    assert obj["custom_id"], f"missing custom_id on line {i}"
    assert obj["custom_id"] not in seen, f"duplicate custom_id {obj['custom_id']}"
    seen.add(obj["custom_id"])
    assert obj["type"] == "completions", f"unexpected type on line {i}"
    assert obj["version"] == "v1"
    body = obj["body"]
    assert "model" in body, f"missing model on line {i}"
print(f"Validated {len(seen)} requests")
```
